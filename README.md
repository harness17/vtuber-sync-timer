# VTuber Live Sync Timer

VTuber同時視聴配信サポートタイマー — HTML単体で動作、外部依存なし

> 本ツールは [Claude](https://claude.ai)（Anthropic）との対話を通じて生成されました。  
> Claude Artifact: https://claude.ai/public/artifacts/a9a520ac-c540-410c-a3f6-07887e600e9d

---

## 概要

HTMLファイルをブラウザで開くだけで使用可能。インストール・サーバー・外部ライブラリ不要。  
VTuberが映像作品（アニメ・映画・特撮など）を流しながら実況する「同時視聴配信」をサポートするタイマーツールです。

---

## 機能一覧

### 作品情報
タイトル・サブタイトル・話数を入力するとアナウンス文を自動生成。手動編集・コピーボタン付き。尺は作品タイマーのDURATION入力と連動して自動反映。

### 再生開始カウントダウン
分・秒を指定してカウントダウン。0になると自動で作品タイマーが起動。残り10秒以下で点滅警告。初期値5秒。

### 作品再生タイマー
経過・残り時間を同時表示。プログレスバー付き。一時停止中は1s / 10s / 30s単位の巻き戻し・早送り可能。LAP記録機能。DURATION入力で終了アラーム。

### CM / 休憩タイマー
90秒・2分・5分・10分のプリセット付きカウントダウン。計測モード（ストップウォッチ）も対応。終了時にビープ音アラーム。

### 配信経過タイマー
配信全体の経過時間を管理。「〇時間〇分〇秒経過」の日本語サブテキスト表示付き。

### パネル背景色カスタマイズ
ヘッダーのPANEL BGから全パネルの背景色を一括変更。カラーピッカー・カラーコード直打ち対応。設定はlocalStorageに保存されリロード後も維持。

---

## 使いかた

1. `vtuber-timer.html` をブラウザで開く
2. 作品情報にタイトル・サブタイトル・話数を入力
3. 作品タイマーの DURATION に作品の尺（時・分・秒）を入力
4. アナウンス文をコピーして視聴者へ告知
5. カウントダウンをスタート → 0秒で作品タイマーが自動起動

---

## 動作環境

モダンブラウザ（Chrome / Firefox / Edge / Safari）対応。  
ローカルファイルとして開く場合、アラーム音にはブラウザの音声許可が必要な場合があります。
